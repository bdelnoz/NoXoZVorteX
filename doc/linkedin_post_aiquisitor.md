🚀 A.I.Quisitor : Transformez vos conversations IA en cartographie de compétences

🇬🇧 English version coming soon!

Après des centaines d'heures avec ChatGPT, Claude et Le Chat, je me suis posé LA question : comment valoriser tout ce savoir-faire accumulé ?

La réponse : A.I.Quisitor 🔍

🎯 Le concept
Un outil open source qui analyse automatiquement vos exports JSON (ChatGPT, Claude, Le Chat, et bientôt Grok) pour en extraire une liste structurée de vos compétences techniques.

✨ Ce qui le rend unique :
✅ Détection automatique du format (--aiall)
✅ Split intelligent à 31 000 tokens
✅ Détection des doublons entre comptes
✅ Analyse locale GRATUITE (--local) ou avec API Mistral
✅ Rapports CSV/TXT prêts pour LinkedIn/CV
✅ 100% open source (Licence MIT)
✅ Optimisé actuellement pour le français

🛠️ Concrètement, il détecte :
• Vos frameworks (LangChain, Hugging Face, TensorFlow, PyTorch)
• Vos techniques (Prompt Engineering, Fine-tuning, RAG)
• Vos domaines (NLP, Computer Vision, MLOps, Python, DevOps...)
• Vos outils et technologies

📊 Mon cas d'usage réel : 
✅ 1538 conversations analysées (tous mes comptes confondus)
✅ Fichiers structurés avec toutes mes compétences par domaines

💡 Pourquoi ?
Nos échanges avec les IA révèlent notre expertise réelle. Ils montrent comment on résout des problèmes, quels outils on maîtrise. Il était temps de rendre ces compétences visibles et valorisables.

🔧 Installation ultra-simple :
./analyse_conversations_merged.py --install
./analyse_conversations_merged.py --exec --local --aiall --fichier *.json
